## 技术栈
### 前端
- **框架**：vue3(组合式api)
- **UI**：shadcn-vue + Tailwind CSS
- **路由**：vue-router 4
- **HTTP客户端**：axios
- **日期处理**：date-fns
- **图标**：lucide-vue-next

## 开发规范
- 用户对前端开发了解较少，所以你在开发时有较高的自主权，包括修改本文档自身！！
- 本项目采用较新的技术栈，如果有不确定的用法，请调用context7工具查阅相关文档。
- 在每次开发之前，请你先了解当前的文件目录结构。
- 如无说明，你不需要编写单元测试。
- 每次代码编写完成后，无需启动项目，也无需进行集成测试或测试接口。
